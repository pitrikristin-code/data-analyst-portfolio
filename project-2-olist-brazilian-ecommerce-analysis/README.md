# Brazilian E-Commerce Performance Analysis (Olist)

## Business Objective
Analyze sales performance, customer behavior, and delivery operations to understand
how logistics performance impacts customer satisfaction.

## Key Business Questions
- How does revenue and order volume evolve over time?
- Which product categories contribute the most to revenue?
- How does delivery time affect customer review scores?
- What proportion of customers are one-time vs repeat buyers?
- Which Brazilian states generate the highest revenue?
